Этот бот получает геолокацию пользователя через данные об ip, полученные через http://ipinfo.io/json.
Далее с использованием геолокации отправляет запрос в openweather и получает данные в виде json, после чего парсит их и выводит пользователю в следующем формате:
Weather - возвращает все данные о текущей погоде, а именно: Локацию пользоваеля, описание текущей погоды, температура(в градусах цельсия), температура по ощущениям(в градусах цельсия),
максимальная температура(в градусах цельсия), минимаьная температура(в градусах цельсия), влажность(в %), облачность(в %), давление(в миллиметрах ртутного столба)
Wind - вовращает данные о ветре, а именно: направление(северный, северо-восточный и т.п), скорость(в м/с).
Daylight hours - возвращает данные о времени восхода и заката солнца в формате %H:%M
pressure - возвращает информацию о текущем давлении(в миллиметрах ртутного столба)
humidity - возвращает данные о текущей влажности в процентах
cloudness - возвращает данные о текущей облачности в процентах
